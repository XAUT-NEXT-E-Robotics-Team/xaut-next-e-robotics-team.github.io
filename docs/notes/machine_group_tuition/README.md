## 轮组简介：
### 现在主流的轮组有4种，分别是全向轮，麦轮，舵轮和混合轮组
### 1.全向轮：
#### 优点：
* 结构简单易维护，稳定性高，机动性强，
* 对于缩小体积比较友好，
* 价格相对成品内嵌麦轮和舵轮来说较为便宜
* 在小陀螺时的功率损耗小；
#### 缺点：
* 平移时有功率损耗，
* 启动和上坡打滑较为严重
* 推荐开源：
#### 东北大学、广东工业大学、东莞理工学院，南京理工大学等；
### 2.麦轮：
#### 优点：
* 结构相对简单，运动更加平滑，稳定性强，
* 飞坡姿态相对较好
* 缺点：
* 平移时有功率损耗，
* 启动和上坡打滑较为严重
* 成品内嵌麦轮价格较为昂贵
* 平移和小陀螺均有功率损耗
* 推荐开源：
#### 上海交通大学2020年步兵机器人，HEXROLL开源内嵌麦轮
### 3.舵轮：
#### 优点：
* 在各种情况下功率损耗较小（理论无损耗）
* 停止状态下有自锁功能（无打滑）
* 结构较为复杂，分为转向电机和驱动电机，
* 缺点：
* 价格相对全向等比较昂贵（大头在电机）
* 重量较重，且对强度要求极高
* 需要搭配超级电容使用，否则在低等级下机器人功率
#### 不足
* 结构相对复杂维护难度大
* 推荐开源：
#### 西安电子科技大学、太原理工大学、华南理工大学、华中科技大学等
### 4.混合轮组：
#### 目前赛场出现的混合轮组基本有：
#### 前麦轮后全向轮，后驱舵轮前置全向，半舵半全向斜置全驱动，半舵半全向斜置舵驱动全向从动
#### 优点：
* 解决全舵无超电功率问题
* 前麦后全向解决上43度坡打滑问题
#### 缺点：
* 悬挂调节较为复杂，需要大量测试
* 推荐开源：
#### 哈尔滨工业大学（半舵哨兵）、山东理工大学（从动全向）、南京航空航天大学（英雄）等
