学习任务一：SolidWorks 的安装及学习。

SolidWorks 是我队机械组主要使用建模软件，需要自行下载，建议下载2024版本

  1. 下载地址 （注：提取码是：2024）

  2. 软件安装教学视频

  3. 学习SolidWorks使用教程(注：简单练手模型自行查看群文档机械部分（自行查看，不要求全做完，目的是熟练掌握solidworks基本使用)

学习任务二：RoboMaster 裁判系统的下载

搜索 RoboMaster 官网，在RoboMaster 产品中选择裁判系统，在当前赛季的硬件产品中找到对应裁判系统部件，选择了解更多，在页面最下方下载STEP格式文件，打开即可。

学习任务三：RoboMaster 官网开源学习

搜索 RoboMaster 官网，点击 RoboMaster 论坛进入社区。在专栏中选择机械开源专栏后，在左侧搜索栏搜索所希望了解的兵种开源文档并且自行下载。

机械开源文档分为两部分，三维图纸和说明文档。三维图纸为总图，注意格式，如果是step格式装配体一定要用solidworks附带软件eDrawings 打开。说明文档为整车的技术方案总结，很多关键技术点和设计参数都在技术文档中有讲述。（建议二者结合参考。）

学习任务四：运用淘宝获取标准零件参数及图纸。

  1. 对于滑环，电机等标准件： 自行询问商家要模型

  2. 对于螺丝螺母等： 自行下载迈迪工具集或者大工程师插件，直接生成图形；

自行上B站或者csdn官网搜索教程下载

初步考核任务：绘制轮组

最低要求画出非内嵌式麦轮轮组或非内嵌式全向轮，通过组长审核（零件完全定义、装配体正常配合，结构合理）。

但是非内嵌轮组逐渐退出RM舞台，建议自选其他轮组，如内嵌式全向轮、内嵌式麦轮、舵轮等。

需要相关信息或遇到问题无法解决可向学长咨询（不懂就问）。

我队现有技术储备为麦轮（内嵌非内嵌）全向轮（内嵌非内嵌）全舵，半舵半全向。

轮组简介：

现在主流的轮组有4种，分别是全向轮，麦轮，舵轮和混合轮组

1.全向轮：

优点：

  • 结构简单易维护，稳定性高，机动性强，
  • 对于缩小体积比较友好，
  • 价格相对成品内嵌麦轮和舵轮来说较为便宜
  • 在小陀螺时的功率损耗小；

缺点：

  • 平移时有功率损耗，
  • 启动和上坡打滑较为严重
  • 推荐开源：

东北大学、广东工业大学、东莞理工学院，南京理工大学等；

2.麦轮：

优点：

  • 结构相对简单，运动更加平滑，稳定性强，
  • 飞坡姿态相对较好
  • 缺点：
  • 平移时有功率损耗，
  • 启动和上坡打滑较为严重
  • 成品内嵌麦轮价格较为昂贵
  • 平移和小陀螺均有功率损耗
  • 推荐开源：

上海交通大学2020年步兵机器人，HEXROLL开源内嵌麦轮

3.舵轮：

优点：

  • 在各种情况下功率损耗较小（理论无损耗）
  • 停止状态下有自锁功能（无打滑）
  • 结构较为复杂，分为转向电机和驱动电机，
  • 缺点：
  • 价格相对全向等比较昂贵（大头在电机）
  • 重量较重，且对强度要求极高
  • 需要搭配超级电容使用，否则在低等级下机器人功率

不足

  • 结构相对复杂维护难度大
  • 推荐开源：

西安电子科技大学、太原理工大学、华南理工大学、华中科技大学等

4.混合轮组：

目前赛场出现的混合轮组基本有：

前麦轮后全向轮，后驱舵轮前置全向，半舵半全向斜置全驱动，半舵半全向斜置舵驱动全向从动

优点：

  • 解决全舵无超电功率问题
  • 前麦后全向解决上43度坡打滑问题

缺点：

  • 悬挂调节较为复杂，需要大量测试
  • 推荐开源：

哈尔滨工业大学（半舵哨兵）、山东理工大学（从动全向）、南京航空航天大学（英雄）等
